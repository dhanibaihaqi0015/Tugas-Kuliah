6.Description List (Daftar Deskripsi)

Html
Hypertext Markup Lenguage Hyper (HTML) adalah bahasa markup yang digunakan untuk membuat dan menyusun struktur halaman web. HTML menggunakan elemen-elemen seperti heading, paragraf, gambar, tautan,table,dan daftar untuk menampilkan konten di browser.
CSS
Cascading Style Sheets (CSS) adalah bahasa script yang digunakan untuk mengatur tampilan dan tata letak elemen dalam web yang dibuat dengan HTML.
JavaScript
JavaScript (JS) adalah bahasa pemograman yang digunakan untuk membuat halaman web lebih interaktif dan dinamis, kita bisa menambahkan animasi, validasi formulir, menangani event (klik,hover), dan banyak lagi.
